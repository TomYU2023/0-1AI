# 一、选题技巧

## 1. 1 三问题法

1. 真问题：可研究、可操作的问题。
2. 小问题：从时间、空间、群体、理论视角上划分的问题。
3. 好问题：基于理论、有依据的问题。


| **标准**   | **正例**                                                         | **反例**                                 | **理由**                                                                                     |
| ------------ | ------------------------------------------------------------------ | ------------------------------------------ | ---------------------------------------------------------------------------------------------- |
| **真问题** | 研究“低成本医疗设备如何在资源有限的地区有效分配和使用”         | 研究“低成本医疗设备的历史发展”         | 研究可操作性强，可以提出具体的解决方案；历史发展虽然有研究价值，但操作性和实际应用价值较低。 |
| **小问题** | 研究“2010年至2020年间非洲低成本医疗设备的发展和影响”           | 研究“全球所有地区低成本医疗设备的发展” | 从时间和地域上划分，具体可操作，研究范围适中；全球范围过大，难以具体操作和分析。             |
| **好问题** | 研究“低成本医疗设备在改善发展中国家医疗服务中的有效性及其机制” | 研究“低成本医疗设备对医疗服务的影响”   | 基于理论，有具体依据，能够深入探讨具体机制；过于宽泛，缺乏具体理论和依据。                   |


## 1.2 AMJ5个标准

1. 重要性：

   - [ ] 该主题是否面临重大挑战（Grand challenges）?
   - [ ] 是否追求大胆的想法，并采用非常规的方法来解决尚未解决的重大问题？
   - [ ] 处理特定文献或研究领域中尚未解决的大型问题，并以超越现有解释的大胆和非常规的方式解决这些问题。
2. 新颖性

   1. [ ] 知识重组的结果，通过在两种文献或学科之间建立桥梁来创造
   2. [ ] 熟悉陷阱：研究者熟悉的主题。很难跳出固定思维。
   3. [ ] 成熟陷阱：研究成熟的主题。很难提出更新的发现。
   4. [ ] 亲近陷阱：与研究者工作接近的主题。容易忽略新颖的方向。
3. 好奇心

   1. [ ] 当主题的主张与读者理所当然的假设相悖时，主题就会很有趣。
   2. [ ] 避免在标题、主题中透露结论。找到一个无法预测结论的主题。
   3. [ ] 写作从结论出发，随后逐步给出依据。
4. 范围

   1. [ ] 研究范围设置不能太小。大了可以删，小了临时做来不及。
   2. [ ] 可以从时间、空间、群体、理论视角上划分的问题。
5. 操作性

   1. [ ] 找到现有理论或概念无法解释的实际现象或问题。现有创新理论通常认为创新需要大量的资本投入和研发支出。然而，在现实中，有些创新是低成本、低投入的，比如低价的人工膝盖和廉价的医疗设备。
   2. [ ] 研究那些在管理实践中存在显著变化和差异，但现有学术语言无法充分解释的现象。研究者可以探讨为什么某些企业在限制性预算下仍能实现高效创新，并提供具体的管理策略来复制这些成功。
   3. [ ] 确保研究结果能够直接转化为管理或组织的具体行动。提供详细的步骤、方法和工具，帮助管理者在实践中应用研究发现。通过研究低成本医疗设备的成功案例，提出如何在有限预算下进行创新的具体步骤和策略，如优化资源配置、建立跨部门协作团队等。

# 1.3 McGahan (2007)5个标准


| **标准**                         | **正例**                                                                               | **反例**                                     | **理由**                                                                                             |
| ---------------------------------- | ---------------------------------------------------------------------------------------- | ---------------------------------------------- | ------------------------------------------------------------------------------------------------------ |
| **反直觉的见解**                 | 研究“低成本医疗设备是否能在不影响质量的情况下提供有效治疗”                           | 研究“高成本医疗设备的效果”（符合传统假设） | 提供与常识性假设相反的见解，引发读者兴趣和思考；高成本设备效果是常识性假设，缺乏吸引力。             |
| **新的和重要实践的效果**         | 研究“利用3D打印技术制造低成本医疗设备的潜力，并评估其实际效果”                       | 研究“改进现有低成本医疗设备的生产工艺”     | 强调新技术（3D打印）的实际效果，展示创新实践的重要性；改进现有工艺缺乏创新性和重要性。               |
| **显示实践中的不一致之处和后果** | 研究“不同发展中国家在采用低成本医疗设备方面的成功与失败案例，并分析其原因和后果”     | 研究“低成本医疗设备在特定国家的普及率”     | 显示实践中的不一致性，并分析其后果，提供深刻的洞见；研究单一国家的普及率缺乏广泛性和深度。           |
| **提出具体理论解释当前情况**     | 提出“资源有限环境中实现低成本医疗设备高效分配的新理论，并验证其在多个案例中的适用性” | 提出“低成本医疗设备市场推广的通用理论”     | 提出新理论，解释和解决当前实际问题，验证其广泛适用性；通用市场推广理论过于宽泛，缺乏针对性和创新性。 |
| **识别标志性现象，开辟新领域**   | 研究“低成本医疗设备在远程医疗中的应用，探索其潜力和未来发展方向”                     | 研究“低成本医疗设备在传统医院中的使用情况” | 识别新现象（远程医疗），开辟新的研究和实践领域，具有前瞻性；研究传统使用情况缺乏新颖性和前瞻性。     |
